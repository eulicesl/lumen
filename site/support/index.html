Lumen Support

Platforms: iPhone and iPad. Minimum OS target in the current release train: iOS 18.0.

This page provides baseline support information for the App Store build and a stable public URL for App Store Connect.

Common support topics

  • App launch and onboarding
  • Apple Intelligence availability on supported hardware
  • Ollama setup and local-network troubleshooting
  • Microphone, image attachment, and document-import permissions
  • Resetting local data and clearing saved context

How local model support works

Lumen can use Apple Intelligence or Foundation Models on supported devices. It can also connect to a separately running Ollama server that you provide, such as one hosted on your Mac or another machine reachable on your local network.

If no reachable local model endpoint is configured, AI-generation features may be unavailable until the app is pointed at a supported endpoint.

Contact and bug reporting

The current public support channel for Lumen is the GitHub repository. Use the issue tracker for bug reports, release issues, and feature requests.
