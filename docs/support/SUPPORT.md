# Lumen Support

## Scope

This document defines the baseline support information that should be published before App Store submission.

## Recommended public support entry

Publish a stable public support page that includes:

- product name: Lumen
- supported platforms and minimum OS requirements
- how local model support works
- known setup requirements for Ollama
- how to report a bug or request a feature
- how to contact the maintainer

## Minimum support topics

- app launch and onboarding
- Apple Intelligence availability requirements
- Ollama setup and local-network troubleshooting
- microphone, photo, camera, and document-import permission usage
- data deletion and reset instructions
- privacy-policy link

## Submission note

App Store Connect requires a public Support URL.
This repository document is a draft support artifact, not the final hosted support page.
