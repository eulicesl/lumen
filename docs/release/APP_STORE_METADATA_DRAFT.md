## Description

Lumen is a privacy-first AI assistant for Apple devices.

Use Apple Intelligence on supported hardware or connect to Ollama on your local network. Your conversations stay on your device or inside your local environment instead of being routed through a vendor-hosted cloud service.

Lumen is built for fast, native workflows:

- chat with local AI models
- switch between Apple Intelligence and Ollama
- search conversation history
- edit and retry from message history
- import documents for local context
- attach images for analysis
- use voice input
- save important memories for future chats
- access built-in agent tools for small utility tasks

Lumen is designed as a real native product, not a wrapper around a web app.
It supports Dynamic Type, accessibility improvements, and reduced-motion behavior across core surfaces.
