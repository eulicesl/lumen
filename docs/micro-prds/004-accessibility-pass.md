# Micro-PRD 004 — Accessibility Pass

## Goal
Raise Lumen to a serious accessibility baseline.

## Scope
- VoiceOver labels/hints
- Dynamic Type review
- Reduce Motion handling for major animated surfaces
- audit chat actions, model picker, onboarding, widgets

## Acceptance Criteria
- core flows are usable with VoiceOver
- Dynamic Type does not catastrophically break layouts
- animation-heavy areas respect motion sensitivity where appropriate

## Complexity
Medium
