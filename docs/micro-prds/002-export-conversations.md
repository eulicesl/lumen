# Micro-PRD 002 — Export Conversations

## Goal
Allow users to export conversations in trustworthy, portable formats.

## Why
Export is a major trust signal for a privacy-first app and important for App Store/product maturity.

## Scope
- JSON export
- Markdown export
- share sheet integration
- basic export versioning decisions

## Acceptance Criteria
- user can export a conversation from the app
- at least JSON and Markdown are supported
- exported data is understandable and reusable

## Complexity
Medium
