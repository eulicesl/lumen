## User Stories

### US4.1 — Home Screen Widgets
> As a user, I want to interact with Lumen directly from my home screen.

**Acceptance Criteria:**
- **Quick Prompt Widget** (small/medium): Tap to open Lumen with a pre-set prompt or blank input
- **Recent Conversation Widget** (medium/large): Shows last conversation preview, tap to continue
- **AI Status Widget** (small): Shows provider status (connected/disconnected) and last activity
- Widgets update via WidgetKit timeline
- Support for iOS 26 persistent widgets
- Interactive widgets: Quick Prompt allows text input directly

### US4.2 — Lock Screen Widgets
> As a user, I want quick AI access from my lock screen.

**Acceptance Criteria:**
- Compact widget showing Lumen icon + last conversation title
- Tap opens directly to that conversation
- Inline accessory widget for quick prompt

### US4.3 — Live Activities & Dynamic Island
> As a user, I want to see AI generation progress without keeping the app open.

**Acceptance Criteria:**
- When a long response is generating, a Live Activity shows progress
- Dynamic Island shows: model name, token count, elapsed time
- Expanded view shows first ~100 characters of response
- Tap to return to full conversation
- Activity ends when generation completes or errors

### US4.4 — Siri Integration
> As a user, I want to ask Lumen questions through Siri.

**Acceptance Criteria:**
- "Ask Lumen [question]" sends prompt to default model, speaks response
- "Start a conversation about [topic]" creates new conversation
- "What was my last conversation?" reads back title and summary
- App Intents with structured results (rich snippet display)
- Siri Suggestions based on frequent prompts

### US4.5 — Spotlight Search
> As a user, I want to find Lumen conversations from system search.

**Acceptance Criteria:**
- All conversations indexed in CoreSpotlight
- Search matches conversation title and message content
- Results show conversation preview with model badge
- Tap result opens directly to that conversation
- Index updates when conversations are created/modified/deleted
- Privacy mode: option to disable Spotlight indexing

### US4.6 — Shortcuts Actions
> As a power user, I want to use Lumen in my Shortcuts automations.

**Acceptance Criteria:**
- **Ask AI** action: Send prompt, receive text response
- **Create Conversation** action: Start new conversation with optional system prompt
- **List Models** action: Return available models
- **Export Conversation** action: Export specified conversation as text
- All actions parameterized and documented
- Works in Shortcuts, Focus automations, and AppleScript (macOS)

### US4.7 — Control Center
> As a user, I want quick access to Lumen from Control Center.

**Acceptance Criteria:**
- Control Center button opens Lumen to new conversation
- Long-press shows recent conversations

```swift
// LumenWidgets/QuickPromptWidget.swift

import WidgetKit
import SwiftUI

struct QuickPromptWidget: Widget {
    let kind = "QuickPromptWidget"

    var body: some WidgetConfiguration {
        StaticConfiguration(kind: kind, provider: QuickPromptProvider()) { entry in
            QuickPromptWidgetView(entry: entry)
                .containerBackground(.fill.tertiary, for: .widget)
        }
        .configurationDisplayName("Quick Prompt")
        .description("Send a quick message to your AI.")
        .supportedFamilies([.systemSmall, .systemMedium, .accessoryCircular, .accessoryRectangular])
    }
}

struct QuickPromptProvider: TimelineProvider {
    func placeholder(in context: Context) -> QuickPromptEntry {
        QuickPromptEntry(date: .now)
    }

    func getSnapshot(in context: Context, completion: @escaping (QuickPromptEntry) -> Void) {
        completion(QuickPromptEntry(date: .now))
    }

    func getTimeline(in context: Context, completion: @escaping (Timeline<QuickPromptEntry>) -> Void) {
        let entry = QuickPromptEntry(date: .now)
        let timeline = Timeline(entries: [entry], policy: .after(.now.addingTimeInterval(3600)))
        completion(timeline)
    }
}
```

```swift
import ActivityKit
import SwiftUI

struct AIGenerationAttributes: ActivityAttributes {
```

### App Intents

```swift
// AppIntents/AskLumenIntent.swift

import AppIntents

struct AskLumenIntent: AppIntent {
    static var title: LocalizedStringResource = "Ask Lumen"
    static var description = IntentDescription("Send a prompt to your AI assistant.")

    @Parameter(title: "Prompt")
    var prompt: String

    @Parameter(title: "Model", optionsProvider: ModelOptionsProvider())
    var model: String?

    static var parameterSummary: some ParameterSummary {
        Summary("Ask \(\.$prompt)") {
            \.$model
        }
    }

    func perform() async throws -> some IntentResult & ReturnsValue<String> & ProvidesDialog {
        let aiService = AIService.shared
        let response = try await aiService.generate(prompt: prompt, modelName: model)
        return .result(value: response, dialog: "\(response)")
    }
}

struct ModelOptionsProvider: DynamicOptionsProvider {
    func results() async throws -> [String] {
        let models = try await AIService.shared.listAllModels()
        return models.map(\.name)
    }
}

// Register shortcuts
struct LumenShortcuts: AppShortcutsProvider {
    static var appShortcuts: [AppShortcut] {
        AppShortcut(
            intent: AskLumenIntent(),
            phrases: [
                "Ask \(.applicationName) \(\.$prompt)",
                "Send \(\.$prompt) to \(.applicationName)",
            ],
            shortTitle: "Ask AI",
            systemImageName: "bubble.left.and.bubble.right"
        )
    }
}
```

### Spotlight Indexing

```swift
// Services/SpotlightService.swift

import CoreSpotlight

actor SpotlightService {
    static let shared = SpotlightService()

    private let index = CSSearchableIndex.default()

    func indexConversation(_ conversation: ConversationSD) async throws {
        let attributeSet = CSSearchableItemAttributeSet(contentType: .text)
        attributeSet.title = conversation.title
        attributeSet.contentDescription = conversation.messages.last?.content ?? ""
        attributeSet.keywords = extractKeywords(from: conversation)
        attributeSet.lastUsedDate = conversation.updatedAt

        let item = CSSearchableItem(
            uniqueIdentifier: conversation.id.uuidString,
            domainIdentifier: "conversations",
            attributeSet: attributeSet
        )

        try await index.indexSearchableItems([item])
    }

    func removeConversation(_ id: UUID) async throws {
        try await index.deleteSearchableItems(withIdentifiers: [id.uuidString])
    }

    func reindexAll(_ conversations: [ConversationSD]) async throws {
        try await index.deleteAllSearchableItems()
        for conversation in conversations {
            try await indexConversation(conversation)
        }
    }
}
```
